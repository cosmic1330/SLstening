# SLstening 產品需求文件 (PRD)

## 專案願景
SLstening 是一個專為台灣股市設計的高效、專業且具備美觀介面的桌面端股票分析應用程式。透過 Tauri 框架結合 React 技術棧，旨在提供極致的資料讀取速度與豐富的技術指標視覺化功能。

## 目標對象
- 台灣股市投資者。
- 需要進行複雜技術分析與智慧選股的交易者。
- 偏好桌面端應用程式而非網頁版工具的使用者。

## 主要功能
1.  **股票追蹤 (Stock Tracking)**：
    - 新增、移除和管理個人自選股清單。
2.  **資料視覺化 (Data Visualization)**：
    - 多種技術圖表：K線、均線 (MA)、隨機指標 (KD)、能量潮 (OBV)。
    - 自定義專屬指標：MJ、MR 指標圖。
3.  **智慧選股 (Schoice)**：
    - 提供圖形化介面建立複雜查詢條件。
    - 支援技術指標與基本面多維度篩選。
4.  **資料管理 (Data Management)**：
    - 自動從網路抓取最新的股票交易資料。
    - 離線優先 (Offline-First) 策略，資料存儲於本地 SQLite。
5.  **使用者系統**：
    - 透過 Supabase 進行使用者認證與資料雲端同步。

## 驗收標準
- 圖表渲染流暢，單次渲染點數控制在效能規範內 (300-500點)。
- 智慧選股查詢反應時間應小於 2 秒。
- 應用程式具備自動更新功能。
- 字體與 UI 元素符合專案設定的高質感與 Premium 調性。
