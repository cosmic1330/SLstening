✅ Ichimoku 一目雲：建議的分析順序
1. 先看雲（Kumo）判斷大方向

目的：先決定市場是多、空、盤整，不浪費時間。

價格在雲上 → 多頭偏多

價格在雲中 → 盤整 / 不交易

價格在雲下 → 空頭偏空

未來雲上升（轉折線 > 基準線） → 預示偏多

未來雲下降 → 預示偏空

👉 雲給的是「大局觀」，其餘都是細節。

2. 看轉折線（Tenkan） vs 基準線（Kijun）

這兩條線是 Ichimoku 的「短期 / 中期動能」。

轉折線上穿基準線 → 多頭訊號（TK Cross）

轉折線下穿基準線 → 空頭訊號

但要注意位置：

TK Cross 發生位置	訊號強弱
雲上	強多
雲中	中性
雲下	強空

👉 方向 + 位置兩者都要看。

3. 看遲行線（Chikou）確認趨勢是否「乾淨」

遲行線是 Ichimoku 最容易被忽略，但超重要。

遲行線在價格上方 → 多頭乾淨

遲行線在價格下方 → 空頭乾淨

遲行線被K棒糾纏 → 盤整、不要進場

👉 遲行線是用來確認市場是否有空間，而不是當進出場訊號。

4. 用基準線（Kijun）判斷支撐 / 壓力

基準線在 Ichimoku 中就是主支撐。

進場後：

多頭時：跌回基準線，通常是買點或止損點

空頭時：碰到基準線，通常是空點或止損點

👉 基準線 = 中期方向，不要和MA20混為一談。

5. 最後才看雲的厚度與轉換訊號

雲的厚薄代表支撐力道：

雲厚：難反轉，支撐強

雲薄：容易反轉，突破容易成功

✨ Ichimoku 一目雲分析完整流程（簡短記憶版）

雲判趨勢

TK Cross 判動能

遲行線判乾淨度

基準線判支撐壓力

雲厚度看反轉可能


## 未來雲（Senkou Span A / B 形成的雲延伸到未來）其實就是 Ichimoku 的「前瞻性趨勢指示器」，用途很廣，不只是看支撐壓力。重點整理如下：

1. 判斷未來趨勢方向

未來雲上升 → 多頭趨勢延續的可能性高

未來雲下降 → 空頭趨勢延續的可能性高

未來雲扁平 / 厚薄不一 → 盤整或趨勢不明

雲的方向比當下價格更早反映「潛在趨勢」。

2. 提供未來支撐與壓力區

價格在雲上 → 雲下緣是支撐

價格在雲下 → 雲上緣是壓力

雲厚 → 支撐/壓力強

雲薄 → 支撐/壓力弱，突破容易

所以你可以用未來雲做「預測價格碰到哪裡容易停或反彈」。

3. 預測反轉點

Senkou Span A 與 Senkou Span B 收斂 → 未來雲變薄 → 潛在反轉區

雲扭曲 / 變換顏色 → 趨勢可能轉向

對交易策略來說，未來雲的變化常比當下 K 線訊號更早給警告。

4. 與其他線組合形成訊號

TK Cross + 未來雲位置 → 信號強弱判斷

例如：TK Cross 在雲上 + 未來雲上升 → 強多頭

TK Cross 在雲下 + 未來雲下降 → 強空頭

總結一句話：

未來雲就是「預測未來趨勢的雲」＋「預判支撐/壓力區」＋「潛在反轉信號」
